
!!
Весь вывод результатов + содержимое ItemWindow и AdditionWindow производится динамически в коде соотв. окон в коде c#.
!!

DataAccess - работа с БД

DataAccess/HarryCarrier - модели объектов, повторяющие и представляющие физические таблицы из БД

DataAccess/Reps - "Репозитории", отвечающие за проведение транзакций с БД: считывание данных и инициализация 
моделей, запись новых кортежей и тд.

DataAccess.DataBaseContext - определяет контекст БД, создает подключение и определение зависимостей с моделями. Строка подключения 
находится в App.config в блоке connectionString.

Dependencies - Содержит главные абстракции проекта. Соделжит интерфейсы для определения функциональностей для 
репозиториев

Helpers - представляет вспомогательные функции. Например экспорт данных в Word.



Окно авторизации + методы управления пользователем - AuthWindow

Фильтрация, поиск, вывод результаттов - MainWindow

ItemWindow - определяет окна для отображения подробного контента объектов.


AdditionWindow - добавление элоементов вкатегории и добавление новых категорий. 

!!
Для обновления списка категорий в данном окне закрой и открой соответствующий раскрывающийся список.
!!


CategoryAdditionWindow - служит для добавлениия кастомной категории.
